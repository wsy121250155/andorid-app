7.getSupportFragmentManager().beginTransaction().replace()
只是将第一个参数代表的元素（layout/fragment）删除，并没有将它的子元素递归删除。

在第二个fragment上加上一张背景图片，遮住前一个的内容：android:background="@drawable/ic_launcher"
并且要在第二个上添加属性，android:clickable="true"，以保证第一个fragment的空间不会再被用户行为触发。

9.监控内存信息：
private void displayBriefMemory() {
		final ActivityManager activityManager = (ActivityManager) getSystemService(ACTIVITY_SERVICE);
		ActivityManager.MemoryInfo info = new ActivityManager.MemoryInfo();
		activityManager.getMemoryInfo(info);
		Log.i("wsy", "系统剩余内存:" + (info.availMem >> 10) + "k");
		Log.i("wsy", "系统是否处于低内存运行：" + info.lowMemory);
		Log.i("wsy", "当系统剩余内存低于" + info.threshold + "时就看成低内存运行");
	}
